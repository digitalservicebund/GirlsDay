Absätze

Absätze werden verwendet, um Textblöcke auf einer Webseite darzustellen. Sie können verwendet werden, um Ideen zu erklären, Geschichten zu erzählen oder Informationen zu teilen.

Listen

Listen werden verwendet, um Elemente auf einer Webseite in einer bestimmten Reihenfolge darzustellen. Es gibt zwei Arten von Listen: ungeordnete Listen und geordnete Listen.

Ungeordnete Liste

  • Element 1
  • Element 2
  • Element 3

Geordnete Liste

  1. Element 1
  2. Element 2
  3. Element 3

Links

Links werden verwendet, um Benutzer auf andere Webseiten oder zu anderen Abschnitten der aktuellen Webseite zu führen.

Google

Bilder

Bilder werden verwendet, um visuelle Elemente auf einer Webseite darzustellen. Sie können verwendet werden, um Produkte zu präsentieren, Orte zu zeigen oder einfach nur eine visuelle Ergänzung zum Text zu bieten.

Formulare

Formulare werden verwendet, um Benutzereingaben zu sammeln. Sie können verwendet werden, um Feedback von Benutzern zu erhalten, Anmeldungen durchzuführen oder andere Arten von Interaktionen zu ermöglichen.







Divisions

Divisions werden verwendet, um verwandte Elemente zu gruppieren. Sie haben keine visuelle Bedeutung, werden aber oft verwendet, um das Styling zu erleichtern.

Kommentare

Kommentare werden verwendet, um Notizen oder Erklärungen im HTML-Code zu hinterlassen. Sie werden nicht auf der Webseite angezeigt.

Tabellen

Tabellen werden verwendet, um Daten auf einer Webseite in einer tabellarischen Form darzustellen.
