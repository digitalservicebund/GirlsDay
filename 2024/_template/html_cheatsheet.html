HTML Elemente - Übersicht

Absätze

Absätze werden verwendet, um zusammengehörige Textblöcke darzustellen. Sie werden durch den <p></p> tag markiert. 

<p>
  Dies ist ein Typoblindtext. An ihm kann man sehen, ob alle
  Buchstaben da sind und wie sie aussehen. Manchmal benutzt man
  Worte wie Hamburgefonts, Rafgenduks oder Handgloves, um Schriften
  zu testen.
</p>

Überschriften

Überschriften (headings) beschreiben den folgenden Inhalt und werden über den <h1>/<h2>/<h3>/... Tag markiert. Die Zahl ist dabei das Level, je kleiner desto wichtiger. 

<h1>Überschrift 1</h1>

Überschrift 1

 
<h2>Überschrift 2</h2>

Überschrift 2

 
<h3>Überschrift 3</h3>

Überschrift 3

Listen

Listen werden verwendet, um Elemente auf einer Webseite in einer bestimmten Reihenfolge darzustellen. Es gibt zwei Arten von Listen: ungeordnete Listen und geordnete Listen.

Ungeordnete Liste

<ul>
  <li>Element 1</li>
  <li>Element 2</li>
  <li>Element 3</li>
</ul>
  • Element 1
  • Element 2
  • Element 3

Geordnete Liste

<ol>
  <li>Element 1</li>
  <li>Element 2</li>
  <li>Element 3</li>
</ol>
  1. Element 1
  2. Element 2
  3. Element 3

Links

Links werden verwendet, um Benutzer auf andere Webseiten oder zu anderen Abschnitten der aktuellen Webseite zu führen. 

<a href="/index.html"> Relativer Link</a>
 Relativer Link 
<a href="https://digitalservice.bund.de/"> Externer Link</a>
 Externer Link

Bilder

Bilder werden verwendet, um visuelle Elemente auf einer Webseite darzustellen. Sie können verwendet werden, um Produkte zu präsentieren, Orte zu zeigen oder einfach nur eine visuelle Ergänzung zum Text zu bieten. 

<img src="bilder/bild1.jpg" alt="Beispielbild" />

Bilder können auch Links auf andere Seiten sein.


<a href="https://digitalservice.bund.de/">
  <img src="bilder/bild1.jpg" alt="Beispielbild" />
</a>

Divisions

Divisions werden verwendet, um verwandte Elemente zu gruppieren. Sie haben keine visuelle Bedeutung, werden aber oft verwendet, um das Styling zu erleichtern. 


<div style="border: 1px solid gray">
  Äußeres div
  <div style="background-color: aqua">Inneres div</div>
</div>
Äußeres div
Inneres div
