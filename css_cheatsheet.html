CSS Cheatsheet

Basics

Mit CSS kann beschrieben werden, wie Elemente aussehen sollen. Dafür schreiben wir sogenannte CSS Klassen:

Syntax einer CSS Klasse

    selector{
      property: value;
      property2: value2;
    }

Einbinden von Styles

Externe Styles

        <link rel="stylesheet" type="text/css" href="/style.css" />

Interne Styles

      <style type="text/css">
        div { color: #444;}
      </style>

Inline Styles

✎
        <tag style="property: value"> </tag>

Selektoren

Ein CSS-Selektor wird verwendet, um HTML-Elemente basierend auf ihrem Elementnamen, ihrer Klasse, ihrer ID, ihren Attributen und mehr auszuwählen. Hier sind einige Beispiele:

  • Element Selektor: h1, p, div, etc.
  • Klassen S: .classname
  • ID selector: #idname
  • Attribute selector: [attribute=value]

Properties and Values

CSS-Properties (Eigenschaften) definieren den Stil eines HTML-Elements. Hier sind einige gebräuchliche Eigenschaften

  • color: legt die Farbe des Textes fest
  • background-color: legt die Hintergrundfarbe eines Elements fest
  • font-size: legt die Größe des Textes fest
  • padding: legt den Innenabstand eines Elements fest
  • margin: legt den Außenabstand eines Elements fest
  • border: legt den Rahmen eines Elements fest

CSS-Werte (Values) werden verwendet, um den Wert einer CSS-Eigenschaft zu definieren. Hier sind einige Beispiele:

  • color: #ffffff;
  • background-color: #0099cc;
  • font-size: 18px;
  • padding: 10px;

Box Modell

Das CSS-Box-Modell beschreibt das Layout eines HTML-Elements. Jedes Element ist von einem unsichtbaren rechteckigen Rahmen umgeben, der aus Innenabstand (Padding), Außenabstand (Margin), Rahmen (Border) und dem eigentlichen Inhalt des Elements besteht.

Das Box-Modell kann mit CSS-Eigenschaften wie padding, margin und border gesteuert werden.

Im Beispiel oben hat das Element eine Breite von 200 Pixeln und eine Höhe von 150 Pixeln. Der padding-Wert beträgt 20 Pixel und erzeugt einen Innenabstand um den Inhalt des Elements. Der margin-Wert beträgt ebenfalls 20 Pixel und erzeugt einen Außenabstand um das Element. Der border-Wert von 1 Pixel definiert die Dicke des Elementrahmens.
